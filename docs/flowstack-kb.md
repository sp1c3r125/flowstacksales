# Flowstack OS Knowledge Base

## Brand summary
Flowstack OS is a bounded automation deployment platform. It sells approved packages, not arbitrary custom workflow chaos.

## Official packages

### Flowstack Lite Kit
- Positioning: Auto-replies + lead capture
- Pricing: ₱25,000 one-time setup
- Optional support: ₱3,500/month
- Best for: low-volume businesses, low-value leads, simple front-end automation
- Limits: 1 channel, FAQ/menu flow, lead capture, staff handoff, 1 reminder follow-up after 24 hours of no reply

### Starter
- Positioning: Auto-replies + basic lead capture
- Pricing: ₱35,000 setup + ₱12,000/month
- Alternate: ₱0 setup + ₱18,000/month on 6-month term
- Best for: 1 channel, simple intake, low-complexity businesses
- Limits: 1 lead source, 1 booking outcome, 1 pipeline, 1 basic follow-up sequence

### Growth
- Positioning: Lead recovery + booking system
- Pricing: ₱95,000 setup + ₱28,000/month
- Alternate: ₱0 setup + ₱44,000/month on 6-month term
- Best for: daily inquiries, stronger follow-up need, booking-focused operations
- Limits: up to 2 lead sources, up to 2 offers/outcomes, up to 2 pipelines, up to 2 sequences, daily digest + weekly summary

### Scale
- Positioning: Multi-channel + routing + control
- Pricing: ₱210,000 setup + ₱55,000/month
- Alternate: ₱0 setup + ₱90,000/month on 6-month term
- Best for: high volume, multi-offer, multi-staff routing
- Limits: up to 4 lead sources, up to 4 offers/outcomes, up to 5 pipelines, up to 4 sequences, staff routing/escalation/alerts

## Core modules
- BookedOS: instant replies, qualification, follow-up, booking, reminders
- ClientFlow: intake, handoff notes, post-service prompts
- OpsHub: visibility, digest reporting, pipeline metrics

## Recommendation logic
- Lite: use for low-volume/simple FAQ + lead capture
- Starter: use for one bounded source/outcome/pipeline
- Growth: use for lead recovery + booking with more than one bounded path
- Scale: use for multi-staff routing, escalation, and broader control

## Rules
- Never promise arbitrary custom automation as the default offer
- Always map prospects into Lite, Starter, Growth, or Scale first
- Custom requests should be scoped as add-ons or separate work
- Setup covers deployment and onboarding
- Monthly covers support, monitoring, and approved-scope maintenance

## Do not promise
- Unlimited channels or offers inside a base package
- Arbitrary workflow uploads
- Freeform AI-built infrastructure
- Live provider rollout without approval and testing
